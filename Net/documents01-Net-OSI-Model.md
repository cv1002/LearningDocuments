## OSI 七层模型

| OSI 七层模型 | TCP/IP 四层模型 | 对应的网络协议                                |
| ------------ | --------------- | --------------------------------------------- |
| 应用层       | 应用层          | HTTP、DNS、FTP、NFS、WAIS、SMTP、Telnet、SNMP |
| 表示层       | 应用层          | TIFF、GIF、JPEG                               |
| 会话层       | 应用层          | RPC、SQL、NFS                                 |
| 传输层       | 传输层          | TCP、UDP                                      |
| 网络层       | 网络层          | IP、ICMP、ARP、RAPP、RIP、IPX                 |
| 数据链路层   | 网络接口层      | FDDI、Frame Relay、HDLC、PPP                  |
| 物理层       | 网络接口层      | EIA/TIA-232、EIA/TIA-499                      |
